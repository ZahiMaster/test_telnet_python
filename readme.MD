# Telnet test

Check a list of servers and ports in a file and telnet to each line.

## Execution

Create a file named "test_list.txt" and add line by line like this:

IP,PORT

Example:

10.0.0.1,22

10.0.0.1,80

10.0.0.2,8080


## Usage

```python
Test_telnet.py
```
